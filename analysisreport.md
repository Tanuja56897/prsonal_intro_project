\# Sales Analysis Report

\##Overview

This report summarizes the analysis performed on 'sales.csv' using 'sales\_analysis.py'.

This program calculates key metric for product sales.



\## Key Metrics

The program calculates:



-product Quantity

-price

-total revenue

-average sale

-highest sale

-best selling product



\## conclusion

The analysis highlights the best-performing products,revenue trends, and average sales.

These insights can help in pricing decisions,inventory management , and improving quality.
